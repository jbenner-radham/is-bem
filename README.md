# is-bem [![NPM version][npm-image]][npm-url] [![Build Status][travis-image]][travis-url] [![Dependency Status][daviddm-image]][daviddm-url]
> Is valid BEM name?

## Installation

```sh
$ npm install --save is-bem
```

## Usage

```js
var isValidBem = require('is-bem');

isValidBem('Rainbow');
```
